# Linkie_MO

Linkie Attendee Mobile App - Ứng dụng di động dành cho người tham gia sự kiện, hỗ trợ chụp ảnh AR và các tương tác thời gian thực.

## Các tính năng chính
- Đăng nhập bằng Google.
- Trải nghiệm chụp ảnh AR với các khung hình độc đáo (AR Frame).
- Nhận thông báo và tương tác trực tiếp tại sự kiện.
